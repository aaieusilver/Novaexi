# Novaexi — MVP 

Plataforma demonstrativa da startup Novaexi, desenvolvida para pitch acadêmico e validação de produto.

## Stack

- Vite 8
- React
- TypeScript
- Tailwind CSS 4
- shadcn/ui (estrutura e componentes locais)
- Radix UI
- Lucide Icons
- React Router com `HashRouter`, compatível com GitHub Pages
